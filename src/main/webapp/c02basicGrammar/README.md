JavaScript
基础语法：
   1.引入方式
        1.内部引入
            <script> js语法  </script>
        2.外部引入
            <script src="外部的js文件">  </script>
   2.注释
        1.单行 //
        2.多行 /* */
        3.变量：统一var来定义
        4.数据类型
            1.数值类型
                var a = 1; var b = 1.2; 
            2.字符串类型
                var a = "你好"; 
            3.布尔类型
                var a = true; 
            4.undefined类型，不含有值
                var a; 
            5.null，空值
                var a = null;
   3.运算符
        1.赋值运算符 =
        2.算术运算符 + - * / % ++ -- += -= *= /= %=
        3.比较运算符 > < == === != !> !< >= <=
        4.逻辑运算符 || && | & 
        5.三目运算符 表达式?值1:值2  
   4.选择与循环语句
        1.if else
        2.switch case break default
        3.for
        4.while/do while
   5.函数
        定义：function 函数名(变量名...){}
        调用：函数名(函数值...)
   6.break/continue
